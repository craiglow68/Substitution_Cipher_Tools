This folder contains several modules able to assist in the encrypting breaking of a shift cipher.

Files: 
	displayPartial.py
	SHIFTREADME.txt
	Shift_Cipher_Report.pdf
	shiftcipher.py
	shiftTools.py
	tokenizer.py

Requires: python 3.0+

usage: python shiftTools.py cipherText
Where cipherText is the text to be decoded.

Notes: shiftTools.py contains the UI and thus is the only file intended to be ran alone. Run in 
directory with other files.
