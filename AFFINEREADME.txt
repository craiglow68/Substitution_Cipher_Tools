This folder contains several modules able to assist in the encrypting and breaking of an affine cipher.

Files:
	Affine_Cipher_Report.pdf
	affinecipher.py
	affineTools.py
	displayPartial.py
	AFFINEREADME.txt
	tokenizer.py

Requires: python 3.0+

usage: python affineTools.py cipherText
Where cipherText is the text to be decoded.

Notes: affineTools.py contains the UI and thus is the only file intended to be ran alone. Run in 
directory with other files.
